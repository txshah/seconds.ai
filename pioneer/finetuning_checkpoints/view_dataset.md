# seconds-ai-legal-data — Training Dataset

300 synthetic consumer complaint examples generated by Pioneer, used to fine-tune `LawClassActionClassifier`.

## Curl command (run this anytime to re-fetch)

```bash
curl -s "https://api.pioneer.ai/felix/datasets/3335e516-8508-4d82-a050-36ac48fad9af/1/download" \
  -H "X-API-Key: $PIONEER_KEY"
```

Or save to file:
```bash
curl -s "https://api.pioneer.ai/felix/datasets/3335e516-8508-4d82-a050-36ac48fad9af/1/download" \
  -H "X-API-Key: $PIONEER_KEY" \
  > seconds-ai-legal-data.jsonl
```

Format: newline-delimited JSON (JSONL). Each row: `{"text": "...", "labels": "<label>"}`.
